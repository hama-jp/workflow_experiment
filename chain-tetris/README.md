# Chain Tetris - 連鎖テトリス

ライン消去後にブロックが落下し、新たなライン消去が発生する「連鎖」システムを搭載したテトリスです。

## 遊び方

`index.html` をブラウザで開くだけでプレイできます。サーバー不要、依存ライブラリなしの単一HTMLファイルです。

## 操作方法

| キー | 操作 |
|------|------|
| ← → | 左右移動 |
| ↓ | ソフトドロップ |
| ↑ | 右回転 |
| Z | 左回転 |
| Space | ハードドロップ |
| C | ホールド |
| P | ポーズ |

## 連鎖システム

このゲームの最大の特徴は**連鎖**です。

1. ラインが揃うと消去される
2. 消去後、残ったブロックが**列ごとに独立して落下**する
3. 落下の結果、新たにラインが揃えば再び消去が発生
4. これが連鎖として繰り返される

連鎖が続くほどスコア倍率が上がります（1連鎖ごとに×2.0）。ブロックの配置を工夫して大連鎖を狙いましょう。

## スコアリング

| 同時消去ライン数 | 基本スコア |
|-----------------|-----------|
| 1 | 100 |
| 2 | 300 |
| 3 | 500 |
| 4 | 800 |

- 基本スコア × 現在レベル × 連鎖倍率（2連鎖で×2、3連鎖で×4、4連鎖で×8...）
- ソフトドロップ: 1マスにつき1点
- ハードドロップ: 1マスにつき2点

## ゲーム仕様

- **フィールド**: 10×20
- **ネクスト表示**: 5個先まで
- **ホールド**: 1回のピース設置ごとに1回使用可能
- **回転システム**: SRS（Super Rotation System）準拠のウォールキック
- **落下速度**: レベルが上がるごとに加速（10ライン消去でレベルアップ）
- **ロックディレイ**: 500ms（移動・回転で最大15回リセット可能）
- **ピース生成**: 7種1巡（Bag System）

## 演出

- 連鎖発生時に画面中央にチェイン数を表示
- 3連鎖以上で画面が振動（5連鎖以上でさらに激しく）
- 連鎖レベルに応じて表示色が変化（黄→橙→赤）
- スコア加算時にポップアップ表示

## 技術仕様

- HTML5 Canvas によるレンダリング
- 外部依存なし（単一HTMLファイル）
- `requestAnimationFrame` によるゲームループ
